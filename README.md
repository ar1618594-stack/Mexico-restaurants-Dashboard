# 🍽️ Restaurant Analytics Dashboard | Power BI

## 📌 Project Overview
This project is a Restaurant Analytics Dashboard built using **Power BI**. The dashboard analyzes restaurant performance, customer ratings, cuisines, and consumer preferences to provide business insights.

---

## 📂 Project Files

| File | Description |
|------|-------------|
| restaurants_data.pbix | Power BI Dashboard |
| restaurants.csv | Restaurant information |
| restaurant_cuisines.csv | Cuisine details |
| ratings.csv | Customer ratings |
| consumers.csv | Consumer demographic data |
| consumer_preferences.csv | Customer food preferences |
| data_dictionary.csv | Dataset column descriptions |
